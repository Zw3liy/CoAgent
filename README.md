# CoAI Agent

[![Agent Status](STATUS.md)](https://github.com/Zw3liy/CoAgent)

## Overview
CoAI Agent is a **CMD-based diagnostic and automation framework** that integrates batch scripts with Python modules.  
It runs health checks, resolves conflicts, generates proposals, automates workflows, and syncs results directly to GitHub with live status badges.

---

## ✨ Features
- **Diagnostics & Repair**
  - Detects port conflicts (Apache/MySQL).
  - Logs agent health and repair actions.
- **Python Intelligence**
  - Troubleshooting (`co_troubleshoot.py`)
  - Fuel cost calculator (`co_fuelcalc.py`)
  - Proposal drafting (`co_proposal.py`)
  - Script generator (`co_scriptgen.py`)
  - SEO optimizer (`co_seo.py`)
  - Badge updater (`co_badge.py`)
- **Status Badge**
  - Auto-updates `STATUS.md` with timestamp + health summary.
  - Displays live badge in README.
- **GitHub Sync**
  - Stages, commits, pulls, and pushes changes automatically.
